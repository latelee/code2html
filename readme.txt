基于LEX的用于HTML的语法高亮工具

///////////////////////////////////////////////////////////
编写目的：
为在网站、博客、论坛上显示自定义的语法高亮着色，使用LEX作词法
分析，将与HTML语言结合，可以在大部分网站上显示。――但也有部分
是不支持的。

放到网站上的代码及Makefile只是为了方便显示，可能与实际的使用有
出入，如Makefile中的命令必须是Tab开头，但网站上的已经被4个或8个
空格代替，因此需要将其改回Tab。

其它没有明确说明的注意事项不再说明。

//////////////////////////////////////////////////////////
三种语法高亮：
1、代码，主要是C/C++
2、makefile文件
3、shell脚本及命令行

////////////////////////////////////////////////////////////
编译及用法：

1、Linux环境：
条件：假设系统中已有lex
编译：$ make -f Makefile-linux
清除：$ make -f Makefile-linux clean

用法：
$ code2html < hello.c > hello.html

2、Windows环境：
条件：cl.exe必须在PATH中，当前目录必须有flex.exe(已有)
编译：DOS> nmake -f Makefile-win32
清除：DOS> nmake -f Makefile-win32 clean

用法：
DOS> code2html.exe < hello.c > hello.html

发布到网站上：
生成html文件后，双击打开，复制到博客或论坛的文章编辑器中相应位置即可。
示例：
在
http://blog.csdn.net/subfate
http://www.latelee.org
上大部分的代码都是用这种方法生成的语法着色的。

已知bug:
1、不能处理连续的Tab，暂时的解决方法是使用文本编辑器将tab替换为4个或8个
空格
2、头文件中的“.”，如stdio.h，会被识别为数字的小数点。
